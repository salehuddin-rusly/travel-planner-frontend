About This Project

The Travel Planner is a full-stack application designed to help users organize their upcoming trips, manage budgets, and track savings goals.

Tech Stack Used:

  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Backend: Node.js, Express.js
  • Database: PostgreSQL
  • Tools: REST API, LocalStorage

Key Features:

  • Full CRUD operations for trip management
  • Real-time search and multi-criteria sorting
  • Dynamic status badges (Upcoming, Ongoing, Completed)
  • Individual "Tabung" savings tracker per trip

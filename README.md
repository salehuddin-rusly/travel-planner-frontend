# Travel Planner - Frontend

This is the frontend application for the Travel Planner project, built as part of the Adnexio Software Engineering Final Project. It provides a user-friendly interface to manage travel itineraries and track savings goals.

## Live Website
**URL:** [https://travel-planner-frontend-2z3i.onrender.com](https://travel-planner-frontend-2z3i.onrender.com)

## Key Features
*   **Trip Management:** Add, edit, and delete trips with real-time updates.
*   **Smart Search:** Search destinations and sort trips by date or budget.
*   **Dynamic Status:** Automatic badges for Upcoming, Ongoing, and Completed trips.
*   **Tabung (Savings Tracker):** Visual progress bar for savings goals (using LocalStorage).

## Tech Stack
*   HTML5 & CSS3 (Custom styling)
*   Vanilla JavaScript (ES6+)
*   Fetch API for Backend Communication

## How to Run Locally
1. Clone the repository.
2. Open `index.html` using a live server (e.g., VS Code Live Server).
3. Ensure the backend API is reachable at the configured `API_URL`.

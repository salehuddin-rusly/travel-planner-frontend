# Travel Planner - Frontend (Final Project)

This is the frontend application for the Travel Planner project, built as part of the **Adnexio Software Engineering Final Project**. It provides a user-friendly interface to manage travel itineraries and track savings goals.

## ✨ Key Features
*   **Full CRUD Management:** Add, edit, and delete trips with real-time updates[cite: 1, 2].
*   **Advanced Search & Sorting:** Search destinations and sort trips by date or budget (High-to-Low / Low-to-High)[cite: 1, 2].
*   **Dynamic Status Badges:** Automatic categorization for Upcoming, Ongoing, and Completed trips[cite: 1, 2].
*   **Tabung (Savings Tracker):** Visual progress bar for individual trip savings goals using LocalStorage[cite: 1, 2].

## 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3 (Custom Glassmorphism Design), Vanilla JavaScript (ES6+)[cite: 1, 2]
*   **Backend Connection:** Fetch API to Node.js/Express.js backend[cite: 1, 2]

## 💻 How to Run Locally
1. Clone this repository[cite: 1, 2].
2. Open `index.html` using a live server (e.g., VS Code Live Server)[cite: 1, 2].
3. Ensure the backend API is reachable at the configured `API_URL`[cite: 1, 2].
